- Vite is a build tool to create React, Angular and etc

# Steps to create ReactJS app:
    1. Run the create vite command
        > npm create vite@latest
        > press y
    2. choose project name as '.' if to be created in the current folder (or) name the project name to create a new folder
    3. Select a framework: React
    4. Select a variant: JavaScript
    5. Install with npm and start now: Yes

- .jsx - a javascript component
- Components should follow UpperCamelCase naming convention
